Что сделано:

    ✅ Приложение на React (сборка Vite)

    ✅ Отображение текущей погоды (температура, описание, влажность, ветер)

    ✅ Прогноз на 7 дней (с группировкой данных)

    ✅ Поиск города по кнопке и по клавише Enter

    ✅ Динамическая смена фона в зависимости от погоды

    ✅ Адаптивная вёрстка (CSS-модули / обычный CSS)

Как запустить:

    Установить зависимости: npm install

    Запустить проект: npm run dev

    Открыть в браузере: http://localhost:5173

Используемые технологии:
React, Vite, OpenWeatherMap API, CSS
